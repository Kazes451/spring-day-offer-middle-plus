# Проблематика
 Система управления рабочим временем сотрудников магазинов с возможностью анализа заданий по всем магазинам компании
# Варианты решений
## Вариант 1:
**Описание:** 
    Централизованный сервис с использованием облачных технологий

**Плюсы:** 
- Легкость в управлении данными 
- Единый доступ к данным
- Простота обновлений

**Минусы:**
- зависимость от Интернет соединения
- При большом количестве магазинов может возникнуть проблема с распределением нагрузки на центральный сервер
- риск потери данных при сбое центрального сервера

## Вариант 2:
**Описание:**
Децентрализованная архитектура с использованием распределенных баз данных

**Плюсы:**
- Устойчивость к сбоям сети
- Скорость доступа к данным
- Простота обновлений

**Минусы:**
- сложность синхронизации данных
- сложность управления данными из-за их распределенности
- требует больше ресурсов для создания и поддержки


# Выбранное решение:
Централизованный сервис с использованием облачных технологий

# Обоснования / преимущества выбранного решения:
 - Мобильные приложения или веб-интерфейс предоставляют сотрудникам доступ к системе для просмотра, создания и обновления задач. 
 - Нет необходимости в содержании ИТ-отдела в каждом из магазинов для поддержки локальных систем
 - Простота в обновлении содержимого задач, анализа данных и управления доступом к ним
 - консистентность данных и возможность просмотра в реальном времени
 - возможность использования персональных устройств сотрудников для получения задач
